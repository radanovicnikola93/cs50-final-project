Nikola's Weather APP
This is a final project for the Harvard's Introduction to Computer Sciense CS50 course.
  • The logo was made in Adobe Photoshop
  • Programming language used: Javascript
  • CSS library: Bootstrap 4
  • All images were downloaded on www.unsplash.com. The author link's are displayed in CSS or HTML comments
  • API Weather provider: OpenWeather Map
  • The website is running on Google Cloud Platform
  • You can check the source code of the project following this link bellow
    Github
Weather Report

Head to Weather Report tab to display current weather information

  • Check current and 3-hour weather report
  • Weather info
  • Pressure
  • Humidity
  • Min / Max temperature
  • Wind speed / Wind direction
  • Cloudiness
Forecast

Head to Forecast tab to forecast weather for desired days

  • Forecast weather for tomorrow and up to 14 days
  • Weather info
  • Pressure
  • Humidity
  • Min / Max temperature
  • Wind speed / Wind direction
  • Cloudiness
Map

Head to Map tab to search on the map for your desired location

  • This is the easiest way to search for multiple locations
  • Just search for your desired location by exploring the map
  • The weather report will pop up on the map as soon as you click the city
  • Zoom in the map for more detailed search
